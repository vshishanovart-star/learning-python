## English

### Repository Overview

This repository contains my solutions to practical assignments from the **first part** of the **"Python Developer from Zero to PRO"** course by Skillbox.

Here, I save my code for future reference, to track my learning progress, and to build a portfolio. The solutions are organized by course modules (topics) into separate folders.

**Author:** Vadim Shishanov (Skillbox student)

### Contents

The `PYTHON_PART_1` folder contains subdirectories corresponding to different course modules. Each folder represents a specific topic and contains `.py` files with task solutions from that module.

### Purpose

1.  **Learning Archive:** To systematically store my work throughout the course.
2.  **Skill Demonstration:** To showcase my understanding of fundamental Python concepts.
3.  **Future Reference:** To have a personal knowledge base I can return to.
4.  **Version Control Practice:** To improve my skills in using Git and GitHub.

### How to Use

*   Navigate to the folder of the topic you're interested in (folders are named by module).
*   Each folder contains `.py` files with task solutions.
*   You can run any script to see the output: `python script_name.py`

### Notes

*   These are **my personal solutions**. They may not be the only or the most optimal way to solve a task.

*   I am open to feedback and suggestions for improving the code.
