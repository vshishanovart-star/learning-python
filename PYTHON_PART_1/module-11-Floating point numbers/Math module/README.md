# Math Module - Skillbox Python Course

## Краткое описание / Description

Эта папка содержит решения задач по Python, посвященные работе со встроенным модулем `math`. Задачи демонстрируют практическое применение математических функций и констант для решения различных вычислительных проблем.

This folder contains Python task solutions focused on working with the built-in `math` module. The tasks demonstrate the practical application of mathematical functions and constants to solve various computational problems.

## Цели / Objectives

*   **Освоить встроенный модуль `math`:** Изучить и применить на практике ключевые функции и константы модуля.
*   **Развить навыки решения математических задач:** Научиться использовать инструменты Python для реализации математических формул и вычислений.
*   **Закрепить основы Python:** Применить на практике базовые конструкции языка (переменные, циклы, условные операторы) в связке с математическим модулем.

*   **Master the built-in `math` module:** Learn and apply key module functions and constants in practice.
*   **Develop mathematical problem-solving skills:** Learn to use Python tools to implement mathematical formulas and calculations.
*   **Reinforce Python fundamentals:** Apply basic language constructs (variables, loops, conditional statements) in combination with the math module.

## Как запустить / How to Run

1.  Убедитесь, что на вашем компьютере установлен Python (версии 3.x).
2.  Скопируйте файлы с расширением `.py` из этой папки в удобное для вас место.
3.  Чтобы запустить любой скрипт, откройте терминал (командную строку), перейдите в папку с файлом и выполните команду:
    ```bash
    python имя_файла.py
    ```
    Например:
    ```bash
    python task_1.py
    ```
